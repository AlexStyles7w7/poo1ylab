## Actividad 1
- _p01_HolaMundo:
Crear un "hola mundo".

- _p02_AreaCirculo:
Crear un programa que calcule el area de un circulo.

Crear un codigo que calcule el area de un triangulo.

Crear un codigo que calcule la paga de un trabajador, disminuyendo los impuestos de raiz.
